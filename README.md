# mmd2evernote

I wanted a way to edit markdown files in vim and upload them to evernote. This is a quick hack to do that.

## How to use

### Creating New Notes

`Enew notebook-name note-name`

### Adding a Note to Evernote

`Esave`

## Dependencies

* AppleScript
* multimarkdown
* Ruby
